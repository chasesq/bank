╔════════════════════════════════════════════════════════════════════════════════╗
║                        REAL-TIME FUNCTIONS                                    ║
╠════════════════════════════════════════════════════════════════════════════════╣
║                                                                                ║
║  1. RECEIPT LOADING                                                           ║
║     ├─ Modal opens                                                            ║
║     ├─ Spinner shows (50ms)                                                   ║
║     ├─ Content loads (300ms total)                                            ║
║     └─ Fade-in animation                                                      ║
║                                                                                ║
║  2. COPY REFERENCE NUMBER                                                     ║
║     ├─ User clicks copy button                                                ║
║     ├─ Reference copied to clipboard (35ms)                                   ║
║     ├─ Checkmark appears on button                                            ║
║     ├─ Toast notification                                                     ║
║     └─ Auto-dismiss after 2 seconds                                           ║
║                                                                                ║
║  3. DOWNLOAD RECEIPT                                                          ║
║     ├─ User clicks download                                                   ║
║     ├─ Receipt text generated                                                 ║
║     ├─ File downloaded as .txt                                                ║
║     ├─ Filename: chase-receipt-[reference].txt                                ║
║     └─ Toast confirmation                                                     ║
║                                                                                ║
║  4. PRINT RECEIPT                                                             ║
║     ├─ User clicks print                                                      ║
║     ├─ HTML receipt generated                                                 ║
║     ├─ Print dialog opens (400ms)                                             ║
║     ├─ Professional formatting                                                ║
║     └─ Toast notification                                                     ║
║                                                                                ║
║  5. SHARE OPTIONS                                                             ║
║     ├─ User clicks share                                                      ║
║     ├─ Dropdown menu opens                                                    ║
║     ├─ Copy to Clipboard:                                                     ║
║     │  └─ Formatted transaction summary                                       ║
║     ├─ Email Receipt:                                                         ║
║     │  ├─ Opens native email client                                           ║
║     │  └─ Pre-filled with receipt                                             ║
║     └─ Send via SMS:                                                          ║
║        ├─ Opens native SMS app                                                ║
║        └─ Transaction summary in message                                      ║
║                                                                                ║
║  6. TOGGLE FAVORITE                                                           ║
║     ├─ User clicks star button                                                ║
║     ├─ Star fills/unfills (150ms)                                             ║
║     ├─ Saved to localStorage                                                  ║
║     ├─ Toast notification                                                     ║
║     └─ Persists across sessions                                               ║
║                                                                                ║
║  7. DISPUTE TRANSACTION                                                       ║
║     ├─ User clicks dispute button                                             ║
║     ├─ Receipt modal closes                                                   ║
║     ├─ Dispute drawer opens                                                   ║
║     ├─ Transaction ID passed to handler                                       ║
║     └─ User can report unauthorized transaction                               ║
